# Backend--CampusMarket
 è il marketplace perfetto per gli studenti universitari che vogliono comprare e vendere oggetti usati. Dai libri ai dispositivi elettronici, dall’arredamento all’abbigliamento, qui puoi trovare tutto ciò di cui hai bisogno a prezzi convenienti, direttamente dalla tua community universitaria. Con ScambioUni, risparmi, guadagni e dai una seconda vita agli oggetti! 🚀
